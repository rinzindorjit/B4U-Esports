# Pi Testnet App

A simple web app integrated with the Pi Testnet for user authentication and payments.

## Prerequisites
- Node.js (v16 or higher)
- Pi Browser (for testing in the Pi Testnet environment)
- Pi Testnet Wallet (created via `wallet.pi` in Pi Browser)
- Pi Testnet API Key (obtained from `develop.pi`)
